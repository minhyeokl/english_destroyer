## 워드 파일에서 영어문단을 없애드립니다.
### 사용법
1. 영어 문단을 삭제할 docx 파일에서 코드에 **코드 스타일**을 적용합니다.
2. 프로그램을 실행합니다.
3. 영어를 지울 docx 파일을 불러옵니다.
4. 스타일 목록에서 영어 문단을 남겨둘 스타일에 체크해둡니다.
5. **영어문단 제거하기** 버튼을 누릅니다
